# Changelog

## 1.0.1

### Added
- **Initial Release**: Official launch of the `duo_icons` package.
- Full collection of modern duotone icons optimized for Flutter.
- **Native Dart Support**: High-performance rendering with full **Null Safety** (Dart 3+).
- **Customizable UI**: Easy control over primary and secondary colors for duotone effects.
- **Cross-Platform**: Verified compatibility for **iOS, Android, and Web**.
- Comprehensive documentation and usage examples.
